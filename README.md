# adv3lite/tads3-utils
Set of adv3lite/TADS3 utilities.  These are intended to add some extension and checking capability.  Some of these have since been folded directly into Eric Eve's adv3lite GitHub repo at

## Check Room Descriptions
`check_descriptions` contains code (and a `README`) that is used to check your room descriptions from a player perspective.  It highlights where a room description mentions object(s) that the player would likely try to eXamine but has no further details.

## Check Game Puzzles
`check_puzzles` contains code (and a `README`) that is used to check that the puzzles in your game can be activated and resolved.  It actually does so by playing the game programatically with guidance as to what puzzles are available and their solution.
